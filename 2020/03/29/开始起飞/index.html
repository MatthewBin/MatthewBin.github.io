HEXO 常用的命令

  1. hexo new “name” # 新建文章
  2. hexo new page “name” # 新建页面
  3. hexo g # 生成页面
  4. hexo d # 部署
  5. hexo g -d # 生成页面并部署
  6. hexo s # 本地预览
  7. hexo clean # 清除缓存和已生成的静态文件
  8. hexo help # 帮助
